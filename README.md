# Telco Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn data from a telecom company to identify key factors affecting customer retention.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Analysis Performed
- Data Cleaning and Preprocessing
- Handling missing values
- Feature transformation
- Exploratory Data Analysis (EDA)
- Customer segmentation based on churn

## 🔍 Key Insights
- Customers with month-to-month contracts have higher churn
- Senior citizens have higher churn percentage
- Customers using electronic check payment method churn more
- Long tenure customers are less likely to churn

## 📁 Dataset
Telco Customer Churn dataset (public dataset)

## 🚀 Conclusion
The analysis helps identify high-risk customers and provides insights for improving customer retention strategies.
